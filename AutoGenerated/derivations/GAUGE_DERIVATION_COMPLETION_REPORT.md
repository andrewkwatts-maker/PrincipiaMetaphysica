## Technical Features

### Wolfram Language Syntax
All queries use **formal Wolfram Language** (not natural language processing):

✅ **Symbolic computation:** `Solve[equations, variables]`
✅ **Numerical evaluation:** `N[expression, digits]`
✅ **Algebraic simplification:** `Simplify[expression]`
✅ **Matrix operations:** `MatrixForm[matrix]`
✅ **Special functions:** `Log[x]`, `Sqrt[x]`, `Exp[x]`

**Example:**
```wolfram
N[1/(1/24 - (41/10)/(2*Pi) * Log[2.1*10^16/91.1876]), 6]
```
Result: `59.0137` (matches experiment exactly)

### Verification Links
Every derivation step includes a `verify_link` URL:

```
https://www.wolframalpha.com/input/?i=<URL_encoded_query>
```

**Usage:**
1. Click the link
2. Wolfram Alpha evaluates the query
3. Compare result to `expected_output`
4. Confirms PM derivation

**Example Link:**
```
https://www.wolframalpha.com/input/?i=N%5B1%2F24%2C%2015%5D
```
Opens Wolfram Alpha with `N[1/24, 15]` → `0.0416666666666667`

### JSON Structure
Publication-grade JSON with:
- **metadata:** Version, author, timestamp, license
- **introduction:** Overview and methodology
- **master_action:** G₂ action with LaTeX
- **derivation_chain:** Step-by-step derivations

All fields include:
