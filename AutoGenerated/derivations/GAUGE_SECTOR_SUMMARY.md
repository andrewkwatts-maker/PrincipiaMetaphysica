## Wolfram Language Syntax Reference

All queries use **formal Wolfram Language** (not natural language):

| Function | Purpose | Example |
|----------|---------|---------|
| `Solve[eqs, vars]` | Symbolic solver | `Solve[{x^2 == 4}, x]` |
| `N[expr, n]` | Numerical eval (n digits) | `N[Pi, 10]` |
| `Simplify[expr]` | Algebraic simplification | `Simplify[(a^2 - b^2)/(a - b)]` |
| `Log[x]` | Natural logarithm | `Log[E]` → `1` |
| `Sqrt[x]` | Square root | `Sqrt[24]` |
| `Exp[x]` | Exponential e^x | `Exp[1]` → `2.71828...` |
| `MatrixForm[m]` | Display matrix | `MatrixForm[{{a,b},{c,d}}]` |
