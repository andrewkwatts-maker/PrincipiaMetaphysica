# Experimental Values Added to theory_output.json

## Summary

Added experimental reference values to 39 parameters in `AutoGenerated/theory_output.json` to enable computed vs. experimental comparison on the website's parameters table.

## New Metadata Fields

Each parameter with an experimental counterpart now includes:

- **experimentalValue**: The measured value from established physics
- **experimentalUncertainty**: Measurement uncertainty (1σ)
- **experimentalSource**: Citation (e.g., "PDG 2024", "NuFIT 6.0", "DESI DR2 2024")
- **relativeDeviation**: |computed - experimental| / |experimental|
- **sigmaDeviation**: |computed - experimental| / uncertainty (when uncertainty > 0)
- **oomAgreement**: Boolean indicating if values agree within 1 order of magnitude
- **oomDifference**: Absolute difference in log10 scale

## Usage in Website

The website's parameters table can now display:

1. **Computed Value**: From PM theory
2. **Experimental Value**: From measurements
3. **Deviation**: Both relative (%) and sigma (σ)
4. **Source**: Citation for experimental value
5. **Color Coding**:
   - Green: < 1σ deviation (excellent)
   - Yellow: 1-3σ deviation (good)
   - Red: > 3σ deviation (needs investigation)
