## Physics Background

### The Baryon Asymmetry Problem

**Observation**: The universe contains matter but almost no antimatter.
- Baryon-to-photon ratio: η_B = (n_B - n_B̄) / n_γ = 6.1 × 10⁻¹⁰ (Planck 2018)
- Requires primordial asymmetry generation (baryogenesis)

### Sakharov Conditions (1967)

For baryogenesis to occur, three conditions must be satisfied:

1. **Baryon number (B) violation**
   - Mechanism: Electroweak sphalerons
   - Rate: Γ_sph ~ α_w^5 T (rapid for T > T_EW ~ 100 GeV)
   - Conserves B-L but violates B+L

2. **C and CP violation**
   - Source: PMNS matrix δ_CP phase
   - PM Prediction: δ_CP = 235° (from G₂ holonomy)
   - Experimental hint: δ_CP ~ 230° (NOvA+T2K 2020)

3. **Departure from thermal equilibrium**
   - Mechanism: Heavy right-handed neutrino decay
   - Condition: Γ_N < H(T ~ M_N)
   - Requires M_N1 > 10^9 GeV (Davidson-Ibarra bound)

### Leptogenesis Mechanism (Fukugita-Yanagida 1986)

**Step 1: Lepton Asymmetry Generation**

Heavy RH neutrinos N_i decay out-of-equilibrium:
```
N₁ → ℓ + H    (leptons)
N₁ → ℓ̄ + H̄   (antileptons)
```

CP asymmetry:
```
ε₁ = [Γ(N₁→ℓH) - Γ(N₁→ℓ̄H̄)] / [Γ(N₁→ℓH) + Γ(N₁→ℓ̄H̄)]
```

**Formula** (Davidson-Ibarra, 2002):
```
ε₁ ≈ (3/16π) × (M₁ m₃)/(v²) × sin(δ_CP) × f(M₂/M₁)
```

where:
- M₁ = lightest RH neutrino mass
- m₃ = heaviest light neutrino mass (~ 0.05 eV)
- v = electroweak VEV = 246 GeV
- δ_CP = CP-violating phase (235° from G₂)
- f(M₂/M₁) ≈ 1 for hierarchical masses

**Step 2: Washout Effects**

Inverse decays and scatterings reduce asymmetry.

Washout parameter:
```
K = Γ_N / H(T=M_N)
```

Efficiency factor κ(K):
- **Weak washout** (K < 1): κ ≈ 0.3
- **Intermediate** (1 < K < 10): κ ≈ 0.1/K
- **Strong washout** (K > 10): κ ≈ 0.01/K^1.16

**Step 3: Sphaleron Conversion**

Electroweak sphalerons convert lepton asymmetry to baryon asymmetry:
```
Y_B = c_sph × Y_L
```

where c_sph = 28/79 (Standard Model value).

**Step 4: Final Baryon Asymmetry**

```
η_B = c_sph × |ε| × κ × D
```

where D is dilution factor (typically ~ 1).

## Technical Notes

### Why M_N1 < 10⁹ GeV Still Works

The standard Davidson-Ibarra bound assumes **strong washout** (K >> 1) and is conservative. In our case:

1. **Weak washout regime**: K ~ 10⁻⁷ << 1
2. **Higher efficiency**: κ = 0.3 (vs. κ ~ 0.01 for strong washout)
3. **Exact CP phase**: δ_CP = 235° provides significant asymmetry
4. **Flavor effects**: Not included but can enhance at lower M_N1

The weak washout regime allows M_N1 ~ 10⁸ GeV while still producing observed η_B.

### Systematic Uncertainties

1. **Neutrino masses**: m₃ ~ 0.05 ± 0.01 eV
   - Affects both ε and κ
   - ±20% uncertainty in M_N1

2. **CP phase**: δ_CP = 235° ± 20°
   - Directly enters ε via sin(δ_CP)
   - ±10% uncertainty in η_B

3. **Washout factor**: Model-dependent
   - Assumes hierarchical RH neutrinos
   - Quasi-degenerate masses give different κ(K)

4. **Flavor effects**: Not included
   - Important for M_N1 < 10¹² GeV
   - Can enhance η_B by factor ~1.5
