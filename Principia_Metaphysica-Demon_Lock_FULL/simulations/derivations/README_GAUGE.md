## Wolfram Language Syntax

All queries use **formal Wolfram Language** for symbolic computation:

### Basic Functions:
- `Solve[equations, variables]` - Symbolic equation solving
- `N[expr, n]` - Numerical evaluation to n digits
- `Simplify[expr]` - Algebraic simplification
- `MatrixForm[matrix]` - Display matrix
- `Log[x]` - Natural logarithm
- `Sqrt[x]` - Square root
- `Exp[x]` - Exponential

### Example:
```wolfram
(* Solve for Betti numbers *)
Solve[{b2 == 14, b3 == 24, chi == b2 - b3 + 2}, {b2, b3, chi}]

(* Numerical evaluation *)
```
