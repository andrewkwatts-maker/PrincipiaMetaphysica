## Usage

### Quick Start

Generate all plots:
```bash
cd /path/to/PrincipiaMetaphysica
python simulations/generate_pm_plots_v16_1.py
```

Validate results:
```bash
python simulations/validate_plots_v16_1.py
```

### Integration with Paper

The plots can be directly included in LaTeX documents:

```latex
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.8\textwidth]{AutoGenerated/plots/neutrino_mixing_contours_v16_1.png}
  \caption{Neutrino mixing parameters: PM v16.1 prediction compared to NuFIT 6.0
           3$\sigma$ confidence regions for Normal Ordering (NO, blue) and
           Inverted Ordering (IO, green).}
  \label{fig:neutrino_mixing}
\end{figure}
```

### Integration with Website

For web display, plots can be included directly:

```html
<img src="AutoGenerated/plots/hubble_tension_v16_1.png"
     alt="Hubble Tension Resolution"
     style="max-width: 100%; height: auto;">
```
