## Rebuild locally

**Windows:**

```bat
build.bat
```

**Any platform:**

```bash
pip install -U metaphysica[full]
python -c "import metaphysica; metaphysica.build(out_dir='.')"
```

Both paths regenerate `AutoGenerated/`, `Pages/`, and all derived assets in place.
