# Validation Suite - Quick Reference

## One-Line Commands

```bash
# Full validation (everything)
python simulations/validate_all.py

# Quick validation (skip simulations, ~5 seconds)
python simulations/validate_all.py --skip-simulations

# Data-only validation (skip sims + tests, <1 second)
python simulations/validate_all.py --skip-simulations --skip-tests

# Quiet mode (minimal output)
python simulations/validate_all.py --skip-simulations --quiet
```

## Exit Codes

- **0** = All passed ✅
- **1** = Something failed ❌

## What Gets Checked

| Check | What | Duration |
|-------|------|----------|
| 1. Simulations | Run all v16 sims | 30-60s |
| 2. Pytest | Unit tests | 5-10s |
| 3. Theory Output | JSON structure | <1s |
| 4. Experimental Data | PDG/NuFIT/DESI files | <1s |
| 5. Formula Metadata | All formulas valid | <1s |
| 6. Parameter Provenance | Source tracking | <1s |
| 7. AutoGenerated JSON | All JSON files | <1s |

## Output

- **Console**: Summary with PASS/FAIL status
- **JSON Report**: `simulations/validation_report.json`

## Common Workflows

### Before Commit
```bash
python simulations/validate_all.py --skip-simulations
```

### After Changing Simulations
```bash
python simulations/validate_all.py
```

### Quick Check
```bash
python simulations/validate_all.py --skip-simulations --skip-tests
```

## CI/CD (GitHub Actions)

- **Triggers**: Push to main/develop, PRs, manual dispatch
- **Runs**: 2 jobs (full validation + quick validation)
- **Python**: 3.11, 3.12
- **Artifacts**: Validation reports, theory output
- **File**: `.github/workflows/validate.yml`

## Troubleshooting

| Issue | Solution |
|-------|----------|
| `theory_output.json not found` | Run `python run_all_simulations.py` |
| `pytest not installed` | `pip install pytest` |
| Timeout | Increase timeout in validate_all.py |
| JSON errors | Check file encoding (UTF-8) |
