V12.8 Simulation: Virasoro Anomaly Cancellation

The critical dimension D = 26 arises from Virasoro anomaly cancellation in bosonic string theory. The central charge must vanish for consistent quantization:

Hover over variables for definitions
$$c_{total} = c_{matter} + c_{ghost} = D + (-26) = 0 \implies D = 26$$

Python Implementation: The following code from virasoro_anomaly_v12_8.py demonstrates the calculation: 

def virasoro_anomaly(D: int = 26) -> Dict:
    """
    Verify Virasoro anomaly cancellation for bosonic string.

    The central charge of the Virasoro algebra must vanish for consistent
    quantization. For the bosonic string:

      c = c_matter + c_ghosts = D + (-26) = 0

    This requires D = 26 spacetime dimensions.
    """
    # Central charge contributions
    c_matter = D           # Each bosonic coordinate contributes +1
    c_ghost = -26          # b,c ghost system (conformal weights 2, -1)
    c_total = c_matter + c_ghost

    # Check anomaly cancellation
    anomaly_free = (c_total == 0)

    return {
        'D': D,
        'c_matter': c_matter,
        'c_ghost': c_ghost,
        'c_total': c_total,
        'anomaly_free': anomaly_free
    }

$$n_{gen} = \frac{|\chi_{eff}|}{48} = \frac{|\chi_{eff}|}{24 \times 2} = \frac{144}{48} = 3$$
The Z₂ factor arises from Sp(2,R) gauge fixing in two-time physics

  • Z₂ acts on spinor chiralities: ΨL ↔ ΨR under t₁ ↔ t₂
  • This identification halves independent spinor DOF, doubling the index divisor
