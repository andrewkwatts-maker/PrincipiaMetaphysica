## A.4 First-Class Constraints

$$\chi_1 = X^M P_M = 0$$ **Phase space orthogonality** (A.4a)

$$\chi_3 = P^M P_M = 0$$ **Massless condition** (A.4c)

### A.4.2 First-Class Property Verification

First-class constraints satisfy:
$$\{\chi_a, \chi_b\} \approx C^c_{ab} \chi_c$$

where $\approx$ means "weakly equal" (equal on constraint surface) and $C^c_{ab}$ are structure constants.

**Poisson bracket calculations**:

$$\{X^M P_M, X^N X_N\} = 2X^N \{X^M P_M, X_N\} = 2X^N X^M \{P_M, X_N\}$$
$$= 2X^N X^M (-\delta^N_M) = -2X^M X_M = -2\chi_2$$ **(A.5a)**

$$\{X^M P_M, P^N P_N\} = 2P^N \{X^M P_M, P_N\} = 2P^N P^M \{X_M, P_N\}$$
$$= 2P^N P^M (\delta_M^N) = 2P^M P_M = 2\chi_3$$ **(A.5b)**

$$\{X^M X_M, P^N P_N\} = 4X^M P^N \{X_M, P_N\} = 4X^M P_M = 4\chi_1$$ **(A.5c)**

All Poisson brackets close on the constraints → **first-class confirmed**.
