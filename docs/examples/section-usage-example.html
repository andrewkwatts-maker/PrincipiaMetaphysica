Paper Sections - Usage Example

This demonstrates how to access and display the migrated paper sections from sections.json

Integration Example

This data structure enables:

  • Dynamic rendering: Load sections on-demand from JSON
  • Search indexing: Index subsection content for full-text search
  • Cross-references: Link formulas to their definitions across sections
  • Navigation: Build table of contents from section hierarchy
  • Progressive disclosure: Show/hide subsections interactively

Sample Code:

// Load sections from sections.json
const response = await fetch('/AutoGenerated/sections.json');
const data = await response.json();
const sections = data.sections;

// Get all subsections from Section 2
const section2 = sections['2'];
const subsections = section2.subsections;

// Find subsection about Pneuma field
const pneumaSubsec = subsections.find(s =>
    s.title.includes('Pneuma')
);

// Access subsection content (array of content blocks)
pneumaSubsec.content.forEach(block => {
    if (block.type === 'paragraph') console.log(block.text);
});

// Or load individual section (lazy loading)
const section4 = await fetch('/AutoGenerated/sections/section-4.json')
    .then(r => r.json());
