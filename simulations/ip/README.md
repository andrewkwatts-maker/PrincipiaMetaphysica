This module provides cryptographic tools to establish and verify priority of discovery for key theoretical predictions in the Principia Metaphysica theory.

## Purpose

- **Timestamping**: Create verifiable timestamps for when predictions were first made
- **Priority Claims**: Establish intellectual property rights for novel theoretical predictions
- **Tamper Evidence**: Use SHA-256 hashing to detect any modifications to predictions
- **Reproducibility**: Enable independent verification of all claims

## Files

### Core Files

- **`discovery_hash_registry.py`**: Main script to generate the hash registry

## Verification Protocol

To verify the discovery hashes:

```python
import json
import hashlib

# Load registry
with open('AutoGenerated/discovery_hashes.json', 'r') as f:
    registry = json.load(f)

# Verify each hash
for entry in registry['discoveries']:
    prediction = entry['prediction']

    # Compute hash
    canonical = json.dumps(prediction, sort_keys=True, separators=(',', ':'))
    computed = hashlib.sha256(canonical.encode('utf-8')).hexdigest()

    # Check match
    assert computed == entry['discovery_hash'], f"Hash mismatch for {prediction['name']}"
    print(f"✓ Verified: {prediction['name']}")
```

## Security Properties

- **Collision Resistance**: SHA-256 is cryptographically secure (2²⁵⁶ possible hashes)
- **Preimage Resistance**: Cannot reverse-engineer predictions from hashes
- **Deterministic**: Same prediction always produces same hash
- **Tamper Detection**: Any change to prediction changes hash completely
