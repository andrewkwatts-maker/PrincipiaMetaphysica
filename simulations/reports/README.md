# Principia Metaphysica Report Generators

This directory contains report generation tools for producing publication-quality supplementary materials from theory_output.json.

## report_generator_v16.py

Comprehensive technical summary report generator designed for journal submission supplementary material.

### Features

- **Complete Parameter Documentation**: All 176+ parameters organized by status (ESTABLISHED, GEOMETRIC, DERIVED, PREDICTED)
- **Formula Analysis**: All 91+ formulas categorized and analyzed with validation status
- **Experimental Comparisons**: Automated comparison of predictions with experimental bounds
- **Statistical Analysis**: Chi-square tests, p-values, and sigma deviations
- **LaTeX Tables**: Publication-ready LaTeX table code for paper appendices
- **Provenance Tracking**: Full metadata and source tracking for reproducibility

### Usage

```bash
# From project root
python simulations/reports/report_generator_v16.py
```

### Output

Generates `AutoGenerated/TECHNICAL_SUMMARY.md` containing:

1. **Executive Summary**: Overview of parameters, formulas, and sections
2. **Statistical Summary**: Parameter breakdown and chi-square analysis
3. **Experimental Comparisons**: Theory vs. experiment with sigma deviations
4. **Parameter Tables**: Complete parameter listings by status
5. **Formula Summary**: All formulas organized by category with validation status
6. **Metadata and Provenance**: Data sources and generation details
7. **LaTeX Tables**: Ready-to-copy LaTeX code for paper appendices

### Requirements

- Python 3.7+
- AutoGenerated/theory_output.json must exist
- No external dependencies (uses only standard library)

### Standalone Execution

The report generator is completely standalone:
- No PMRegistry required
- No simulation framework dependencies
- Reads directly from theory_output.json
- Outputs markdown with embedded LaTeX

### Customization

Edit the class parameters to customize:

```python
generator = TechnicalReportGenerator(
    theory_output_path="path/to/theory_output.json",
    output_path="path/to/output.md"
)
generator.run()
```
