## To Compile PDF

### Install LaTeX (if not already installed)
- **Windows:** Download MiKTeX from https://miktex.org/
- **macOS:** Download MacTeX from https://www.tug.org/mactex/
- **Linux:** `sudo apt install texlive-latex-base texlive-latex-extra`

### Compile Command
```bash
cd h:/Github/PrincipiaMetaphysica/submission
pdflatex cover-letter.tex
```
