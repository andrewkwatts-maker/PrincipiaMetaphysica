## Verification Quick Start

### Step 1: Verify Discovery Hashes (5 minutes)

```bash
# Clone repository
git clone https://github.com/[username]/PrincipiaMetaphysica.git
cd PrincipiaMetaphysica

# Run hash verification
python3 << 'EOF'
import json
import hashlib

with open('zenodo_package/certificates/discovery_hashes.json', 'r') as f:
    registry = json.load(f)

print("Verifying discovery hashes...\n")
for entry in registry['discoveries']:
    prediction = entry['prediction']
    canonical = json.dumps(prediction, sort_keys=True, separators=(',', ':'))
    computed = hashlib.sha256(canonical.encode('utf-8')).hexdigest()
    stored = entry['discovery_hash']
    match = "✓ VERIFIED" if computed == stored else "✗ MISMATCH"
    print(f"{match}: {prediction['name']}")
EOF
```

**Expected:** All 10 predictions show "✓ VERIFIED"
